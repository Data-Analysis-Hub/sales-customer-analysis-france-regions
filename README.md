# Brief 2026/02/16
# Tableau de Bord Commercial – Analyse des Ventes

Un **dashboard interactif Excel** complet pour le suivi des performances commerciales :  
chiffre d’affaires, panier moyen, délais de livraison, satisfaction client, retours, efficacité des canaux marketing, etc.

## 🎯 Objectif du projet

Créer un outil d’aide à la décision moderne et actualisable permettant de :

- Consolider plusieurs années de données de ventes provenant de fichiers Excel distincts  
- Nettoyer et structurer les données de manière fiable  
- Calculer et visualiser les **KPI** clés de l’activité commerciale  
- Offrir une navigation intuitive grâce à des **segments dynamiques**  
- Être facilement actualisable quand de nouvelles données arrivent

## ✨ Fonctionnalités principales

- **Consolidation multi-années** via Power Query  
- **Nettoyage avancé** : suppression des doublons, uniformisation des formats, gestion des valeurs manquantes/incohérentes  
- **Indicateurs clés calculés** :
- Chiffre d’affaires total & évolution temporelle
- Panier moyen (montant & quantité)
- Meilleur produit (CA)
- Ventes par produit / région / source de trafic
- Taux de livraison à temps
- Taux de retours produits
- Taux de satisfaction client
- **Visualisations interactives** :
- Courbes d’évolution
- Barres / histogrammes stylisés
- Graphiques KPI simplifiés (style « card »)
- Mise en forme soignée : couleurs harmonisées, icônes, hiérarchie visuelle
- **Filtres dynamiques** (segments slicers) :
- Année
- Produit
- Région
- Source de trafic
- **Actualisation automatique** : ajout d’un nouveau fichier → Rafraîchir → Tout se met à jour

## 🛠️ Technologies utilisées

- **Microsoft Excel** (version récente – Microsoft 365 recommandée)
- **Power Query** (ETL : extraction, transformation, chargement)
- **Tableaux croisés dynamiques** (TCD / PivotTables)
- **Graphiques Excel modernes** + mise en forme conditionnelle & formes
- **Segments (Slicers)** pour l’interactivité
